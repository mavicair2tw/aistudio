自動 PPT 系統

結合 OpenClaw AI 工作流，自動產生簡報大綱、稿件、版型與圖像，適合日常簡報、產品推介、教育訓練。

核心功能

  • 輸入需求（主題、受眾、頁數）→ 產生簡報結構與要點
  • 自動撰寫每頁講稿，可指定語氣/語言
  • 整合圖像生成（情境圖、流程圖）
  • 匯出 Markdown / PPTX / Google Slides API

流程

  1. 收集需求 → 記憶蒸餾保留長期偏好
  2. 技能凍結：版型套用、段落生成、圖像指令
  3. 模型降級：草稿用 gpt-4o-mini，最終審稿才用大型模型
